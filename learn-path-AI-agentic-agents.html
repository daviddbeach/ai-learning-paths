Week 1
LLM Foundations & Your First API Call
▾
Understand what LLMs actually are
Read Andrej Karpathy's "Intro to LLMs" (YouTube, 1hr). Understand tokens, context windows, and temperature.
Concept
Set up your dev environment
Install Python 3.11+, VS Code, and get API keys for Anthropic Claude (or OpenAI). Install the SDK: pip install anthropic.
Setup
Make your first API call
Write a 10-line Python script that sends a message to Claude/GPT-4 and prints the response. No frameworks yet.
Code
Learn prompt engineering basics
System prompts, user prompts, few-shot examples. Try Claude's prompt library and docs.claude.com engineering guide.
Concept
Build a simple chatbot with memory
Extend your script to maintain a conversation history array. Pass prior messages back each turn. See how context works.
Project
Understand structured outputs & JSON mode
Make the LLM return structured JSON. Use it to extract data from text — the bedrock skill of all agent workflows.
Code
Week 2
Tools, Function Calling & RAG
▾
Learn tool use / function calling
Understand how LLMs can call external functions. Read Anthropic's tool use docs and the OpenAI function calling guide.
Concept
Give your LLM a web search tool
Wire up a Tavily or Serper search API as a tool. Let the LLM decide when to search and process results.
Code
Build a tool-calling agent loop
Write a ReAct-style loop: LLM reasons → picks a tool → gets result → reasons again. This is the core agent pattern.
Project
Understand embeddings & vector search
Learn what embeddings are and how semantic similarity works. Use OpenAI or Cohere embeddings API on sample text.
Concept
Build a simple RAG pipeline
Load a PDF or docs, chunk them, embed + store in ChromaDB or FAISS, then retrieve relevant chunks at query time.
Project
Try LangChain or LlamaIndex basics
Skim their quickstart guides. Understand what abstraction they provide (and when to skip them and go raw).
Framework
Week 3
Multi-Agent Systems & Orchestration
▾
Understand multi-agent architectures
Study orchestrator-subagent patterns. Read Anthropic's "Building Effective Agents" guide — it's the best conceptual overview.
Concept
Build an orchestrator + 2 subagents
Create a main agent that delegates tasks to a "researcher" agent and a "writer" agent. Pass outputs between them.
Project
Explore LangGraph or CrewAI
Pick one framework for multi-agent orchestration. Run their official tutorials to see how state graphs and crew roles work.
Framework
Add agent memory (short + long term)
Implement short-term (within session) and long-term memory (persisted to DB/file). Explore the Mem0 library as an option.
Code
Understand MCP (Model Context Protocol)
Read Anthropic's MCP spec. Understand how it standardizes tool/server connections for agents across applications.
Concept
Handle errors, retries & guardrails
Learn to make agents resilient: hallucination checks, retry logic, output validation with Pydantic, and safe stopping conditions.
Code
Week 4
Ship Your First Real Agent App
▾
Design your capstone project
Pick a use case relevant to your work (e.g. SAP config validator, document analyzer, process automation agent). Define scope.
Design
Build the core agent pipeline
Combine everything: tool use, RAG, orchestration. Build the working MVP with basic I/O — no UI needed yet.
Project
Add a simple UI with Streamlit or Gradio
Wrap your agent in a web UI in under an hour. Streamlit is fastest — pip install streamlit, 20 lines of code.
Code
Evaluate and test your agent
Run 20+ test cases. Track accuracy, latency, and cost. Use LangSmith or a simple log file for tracing.
Testing
Deploy to the cloud
Deploy your app to Hugging Face Spaces (free), Railway, or Render. Get a shareable link you can demo to stakeholders.
Deploy
Document & present your agent
Write a one-page brief on what the agent does, how it works, ROI potential, and what to build next. Share with leadership.
